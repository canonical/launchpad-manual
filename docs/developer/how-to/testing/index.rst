Testing
=======

Launchpad relies on a large test suite to keep the platform working as it
grows. These guides cover the practical tasks involved in testing your changes:
running the existing tests, writing new ones, and measuring how much of the
code they cover.

This section is being built up incrementally. Planned guides:

- **Running tests** — run the suite locally, select specific tests and layers,
  and speed up runs.
- **Writing tests** *(planned)* — author Python unit tests, doctests, page
  tests, CLI script tests, and JavaScript tests.
- **Measuring coverage** *(planned)* — generate and read test coverage reports.

For help debugging a failing test, see the :ref:`debugging guides
<how-to-debug>`.
