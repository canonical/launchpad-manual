Create a PPA
============

Prerequisites
-------------

- A Launchpad account.
- For a team PPA, membership of the team with sufficient privileges to
  manage the team's archives.

Steps
-----

1. Open the profile page at ``/~/`` or the team's overview page.
2. Click the "Create a PPA" link, then provide a name and display name
   and submit.

Result
------

The PPA appears on the owner's page and is ready to receive its first
upload. The signing key is generated on the first upload; see
:ref:`ppa-signing-key`.
